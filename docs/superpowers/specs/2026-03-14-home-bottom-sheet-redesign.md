# 홈 바텀시트 접힌 상태 재설계

## 배경

플로팅 캡슐 네비게이션 도입 후, 홈 화면 바텀시트의 접힌 상태(30%)가 플로팅 네비와 시각적으로 충돌한다. 바텀시트가 주간 캘린더 + 할일 카운트를 모두 표시하면서 플로팅 네비와 겹치고, 두 레이어의 배경색이 비슷해 깊이감이 모호하다.

## 결정 사항

**접근 방식:** 접힌 상태 콘텐츠를 컴팩트 카드로 재설계 (방안 B)

### 접힌 상태 (Collapsed) — 컴팩트 카드

- 주간 캘린더 스트립 제거
- "오늘의 할 일" 텍스트 + 완료 카운트(N/M 완료) + 프로그레스 바 표시
- 플로팅 네비 바로 위에 위치하도록 `minChildSize` 조정
- 배경: `AppColors.spaceSurface` + 글래스모피즘 테두리 (플로팅 네비와 동일 스타일)
- 탭하면 바텀시트 펼침

### 펼친 상태 (Expanded) — 변경 없음

- 기존 월간/주간 캘린더 + 날짜별 할일 목록 그대로 유지

### 인터랙션

- 카드 탭 → 바텀시트 0.85로 펼침
- 위로 드래그 → 기존 DraggableScrollableSheet 동작 유지
- 홈 탭 재탭 → 바텀시트 접힘 (기존 동작 유지)

## 변경 범위

### 수정 파일

1. **`home_screen.dart`** — `_buildCollapsedSheet()` 메서드 재작성
   - 주간 캘린더(`SpaceCalendar` compact) 제거
   - 컴팩트 카드 위젯으로 교체: 할일 카운트 + 진행률 바
   - `minChildSize` / `initialChildSize` 값 조정 (0.30 → ~0.18 수준)
   - `snapSizes` 조정

2. **`todo_provider.dart`** (기존 provider 활용 확인)
   - 오늘 할일 총 개수 / 완료 개수 provider가 이미 있는지 확인
   - 없으면 `todoCompletionStatsProvider` 추가 (completed count / total count)

### 신규 파일 없음

- 컴팩트 카드는 `_buildCollapsedSheet` 내부에서 직접 구성 (별도 위젯 불필요)

## UI 스펙

### 컴팩트 카드 레이아웃

```
┌─────────────────────────────────────┐
│  DragHandle                         │
│                                     │
│  📋 오늘의 할 일         2/5 완료   │
│  ━━━━━━━━━━━━░░░░░░░░░░░░░░░░░░░   │  ← 진행률 바
│                                     │
└─────────────────────────────────────┘
   ┌── 🏠 ── ⏱ ── 🧭 ── 👥 ── 👤 ──┐
   └──────────────────────────────────┘
```

### 진행률 바 스타일

- 높이: 4h
- 배경: `Colors.white.withValues(alpha: 0.08)`
- 채움: `AppColors.primary` → 약간 밝은 그라데이션
- 라운드: `AppRadius.small` (4r)
- 할일 0개일 때: 진행률 바 숨김, "할 일을 추가해보세요" 텍스트

### 바텀시트 사이즈 조정

- `initialChildSize`: 0.30 → 0.18 (컴팩트 카드 높이에 맞춤)
- `minChildSize`: 0.30 → 0.18
- `maxChildSize`: 0.85 (유지)
- `snapSizes`: [0.30, 0.85] → [0.18, 0.85]

### 애니메이션

- 카드 탭 시 `animateTo(0.85)` — duration 400ms, `Curves.easeOutCubic` (기존 동일)
- `_isSheetExpanded` 판별 임계값: 0.4 → 0.3으로 조정

## 테스트 계획

- 접힌 상태에서 플로팅 네비와 겹치지 않는지 시각 확인
- 할일 0개 / 1개 / 다수 상태별 카드 표시 확인
- 카드 탭 → 펼침 동작 확인
- 드래그로 펼침/접힘 동작 확인
- 펼친 상태에서 기존 캘린더 + 할일 기능 정상 동작 확인
