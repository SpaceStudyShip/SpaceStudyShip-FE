## [1.0.13] - 2026-01-09

**PR:** #3  

**새로운 기능**
- Firebase 클라우드 메시징(FCM) 기반 푸시 알림 지원 추가
- Toss UX 디자인 원칙을 따르는 새로운 UI 컴포넌트 시스템 도입 (버튼, 텍스트 필드, 카드, 대화상자, 로딩 표시기, 스켈레톤, 스낵바, 빈 상태)
- 자동 포맷팅 기능이 있는 향상된 텍스트 입력 필드
- Android Play Store 배포 자동화 마법사 도구 추가
- iOS 백그라운드 알림 지원 추가

**개선사항**
- 앱 이름을 "우주공부선"으로 한글화
- 향상된 Firebase Crashlytics 통합
- 기기 정보 및 FCM 토큰 관리 서비스 추가
