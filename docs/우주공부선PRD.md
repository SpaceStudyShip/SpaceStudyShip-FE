# 🚀 PRD: 우주공부선 V3

> 공부 시간을 연료로, 우주를 탐험하는 게이미피케이션 Todo + Timer 서비스
> 

---

## 1. 프로젝트 개요

### 프로젝트명

우주공부선 

### 한 줄 정의

> "공부 시간을 연료로, 우주를 탐험하는 게이미피케이션 Todo + Timer 서비스"
> 

### 문제 정의

| 문제 | 설명 |
| --- | --- |
| 동기부여 부족 | 기존 Todo/Timer 앱은 단순 기록에 그쳐 지속 사용 동기가 약함 |
| 시각적 보상 부재 | 누적된 노력에 대한 가시적 성취감 제공이 미흡 |
| 사회적 연결 부재 | 혼자 공부하는 외로움, 함께하는 동기부여 필요 |
| 장기 목표 설정 어려움 | 큰 목표를 작은 단위로 쪼개 달성하는 구조 부재 |

### 솔루션

공부/작업 시간을 **연료**로 전환하여, 국내 → 해외 → 우주로 확장되는 탐험 시스템 제공. 친구/그룹과 함께 경쟁하고 협력하는 소셜 기능 탑재.

### 주요 사용자군

| 사용자군 | 비율(예상) | 특징 |
| --- | --- | --- |
| 대학생 | 45% | 시험 준비, 학점 관리 |
| 취준생 | 25% | 자격증, 코딩테스트 준비 |
| 직장인 | 20% | 자기계발, 업무 집중 |
| 고등학생 | 10% | 수능/내신 준비 |

## 2. 사용자 인터뷰 질문

### A. 현재 습관 파악

- 현재 공부/작업 시간을 어떻게 관리하고 계신가요?
- Todo List 앱을 사용해본 적이 있나요? 어떤 점이 좋았고, 어떤 점이 아쉬웠나요?
- 타이머 앱(뽀모도로 등)을 사용해본 적이 있나요? 지속 사용하셨나요?
- 공부/작업 중 가장 큰 방해 요소는 무엇인가요?

### B. 동기부여 요인 탐색

- 공부할 때 어떤 보상이 있으면 더 열심히 할 것 같나요?
- 게이미피케이션 요소(레벨업, 뱃지, 랭킹 등)가 동기부여에 도움이 된다고 생각하시나요?
- 친구나 스터디 그룹과 함께할 때 더 집중이 잘 되시나요?
- 장기적인 목표(예: 100시간 달성)를 설정하면 도움이 될까요?

### C. 제품 컨셉 검증

- "공부 시간이 연료가 되어 우주를 탐험한다"는 컨셉이 매력적으로 느껴지시나요?
- 국내 도시 → 해외 국가 → 우주 행성으로 확장되는 구조에 대해 어떻게 생각하시나요?
- 친구들과 그룹 랭킹을 보며 경쟁하고 싶으신가요?
- 이 서비스에 월 얼마까지 지불할 의향이 있으신가요?

### D. 개인화 추천 관련

- 앱이 "지금 이 시간대에 집중력이 높아요"라고 알려주면 도움이 될까요?
- 비슷한 목표를 가진 사람들이 많이 하는 Todo를 추천받고 싶으신가요?
- 나와 공부 패턴이 비슷한 사람을 친구로 추천받으면 어떨 것 같나요?

## 6. 유저 플로우 (User Flow)

### A. 신규 가입 플로우

```
[시작] → 스플래시 화면 → 온보딩 (3단계 스와이프) → 로그인 방식 선택
├── 구글 로그인
├── 카카오 로그인
└── 애플 로그인
    → 닉네임 설정 → 목표 설정 (선택) → [메인 홈 진입]

```

### B. 핵심 기능: Todo + Timer 플로우

```
[메인 홈] → "미션 추가하기" 버튼 → Todo 내용 입력 → "추가" 버튼
→ Todo 목록에 표시 → Todo 항목 선택 → "집중 시작" 버튼
→ 타이머 실행 중 (우주선 애니메이션) → 종료 버튼
→ 결과 화면 (집중 시간, 획득 연료, 정각 보너스 여부) → 메인 홈 복귀

```

### C. 탐험 지도 플로우

```
[하단 네비게이션 - 지도 탭] → 탐험 지도 화면 → 현재 위치 + 해금된 장소 표시
→ 잠긴 장소 클릭 → "연료 n통 더 필요해요" 팝업
→ 연료 충분 시 "해금하시겠습니까?" → [해금] → 새 장소 획득 애니메이션
→ (장소에 따라) 우주선 스킨 해금

```

### D. 친구 플로우

```
[프로필 탭] → 친구 관리 → 친구 목록 화면
├── 친구 검색 (닉네임/코드)
├── 친구 요청 보내기
├── 받은 요청 수락/거절
└── 친구 프로필 보기 → 친구 랭킹 확인

```

### E. 그룹 플로우

```
[하단 네비게이션 - 그룹 탭] → 그룹 목록 화면
├── 내가 만든 그룹 (최대 3개)
└── 참여 중인 그룹 (무제한)

[그룹 생성]
→ "새 그룹 만들기" → 그룹명 입력 → 생성 완료 → 초대 코드 발급

[그룹 참여]
→ "초대 코드로 참여" → 코드 입력 → 참여 완료

[그룹 상세]
→ 그룹 클릭 → 그룹 랭킹, 멤버 목록, 그룹 통계

```

### F. 미션 플로우

```
[하단 네비게이션 - 미션 탭] → 미션 목록 화면
├── 일일 미션 (3/5 완료)
├── 주간 미션 (2/4 완료)
├── 월간 미션 (0/2 완료)
└── 히든 미션 (?/?)
    → 미션 완료 시 → 보상 획득 팝업 (연료, 부스터 등)

```

### G. 컬렉션 플로우

```
[프로필 탭] → 뱃지/우주선 컬렉션
├── 뱃지 목록 (획득/미획득)
│   → 대표 뱃지 설정 (최대 3개)
└── 우주선 목록 (정적/Rive)
    → 대표 우주선 설정
    → Rive 우주선 미리보기 재생

```

| 탐험 | 장소 해금 | 국내 완주, 아시아 정복, 지구 마스터, 태양계 정복 |
| 소셜 | 함께하기 | 첫 친구, 그룹 창설자, 인기인 (친구 50명) |
| 히든 | 숨겨진 업적 | 조건 비공개, 달성 시 서프라이즈 |

### 뱃지 등급

```
일반 (회색 테두리)
  ↓
희귀 (파란색 테두리)
  ↓
에픽 (보라색 테두리)
  ↓
레전더리 (금색 테두리 + 빛나는 효과)
  ↓
히든 (무지개색 + 특수 효과)

```

### 10-4. 우주선 스킨 시스템

### 스킨 유형

| 유형 | 설명 | 희귀도 |
| --- | --- | --- |
| **정적 스킨** | 이미지, 움직임 없음 | 일반 ~ 희귀 |
| **애니메이션 스킨 (Rive)** | 실시간 움직임, 이펙트 | 에픽 ~ 레전더리 |

### 스킨 등급별 예시

| 등급 | 유형 | 예시 | 획득 방법 |
| --- | --- | --- | --- |
| 일반 | 정적 | 기본 색상 변형 | 가입, 초반 미션 |
| 희귀 | 정적 | 국가 테마 (일본, 미국) | 해당 장소 해금 |
| 에픽 | Rive | 화성 탐사선 (엔진 불꽃) | 우주 장소 해금 |
| 레전더리 | Rive | 은하 우주선 (별가루 파티클) | 1000시간 달성 |
| 히든 | Rive | 블랙홀 우주선 (왜곡 이펙트) | 히든 미션 |

### Rive 애니메이션 활용

| 상황 | 애니메이션 |
| --- | --- |
| 타이머 대기 | idle 모션 (살짝 떠있는 느낌) |
| 타이머 진행 중 | 엔진 가동 (불꽃, 연기) |
| 부스터 활성화 | 강화 이펙트 (불꽃 커짐) |
| 1시간 달성 | 축하 이펙트 (반짝임) |
| 타이머 종료 | 착륙 애니메이션 |
